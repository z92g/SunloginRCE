# SunloginRCE

# 1.简介

Sunlogin 远程命令执行漏洞 POC&EXP

# 2.用法

```
SunloginRCE -h 192.168.3.222 -p 49664,49665,49666,49667,49668,49669,49674,49819,50246,25,110,135,139,44,5040,7680,7880  // 扫描模式，多个端口用,隔开
SunloginRCE -h 192.168.8.4:49819 -m exp // EXP模式
```
